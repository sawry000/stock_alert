lock Token จะถูกเก็บใน localStorage ของเบราว์เซอร์เท่านั้น
ไม่ถูกส่งไปที่อื่นนอกจาก GitHub API — ใช้ Token ที่มี scope repo เท่านั้น
key วิธีสร้าง Personal Access Token
1. ไปที่ GitHub.com → คลิก profile icon มุมบนขวา → Settings
2. เลื่อนลงไปที่ Developer settings (ซ้ายล่างสุด)
3. Personal access tokens → Tokens (classic) → Generate new token
4. ตั้งชื่อ เลือก Expiration → tick repo → Generate token
5. Copy token มาวางด้านบน (จะเห็นครั้งเดียว)
